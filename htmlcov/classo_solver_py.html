261 with the rule: 

262 

263 False False False = R1 

264 

265 True False False = R2 

266 

267 False True False = R3 

268 

269 True True False = R4 

270

282 Default value : True 

283 

284 classification (bool) : True if the formulation of the problem should be classification (if yes, then it will not be concomitant). 

285 Default value : False 

286 

287 rho (float) : Value of rho for R2 and R4 formulations. 

288 Default value : 1.345 

289 

290 scale_rho (bool) : If set to True, it will become
